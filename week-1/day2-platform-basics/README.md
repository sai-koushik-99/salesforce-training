**What is Salesforce Platform?**
Salesforce is a cloud-based CRM platform that helps businesses manage customer data, sales, services, and business processes in one place. It allows both admins and developers to build applications using configuration tools and coding.

**App :**
An App in Salesforce is a collection of related tools, tabs, and objects created for a specific business purpose. Apps help users access everything they need from one place.

**Examples:**
Sales App
Service App
College Management App

**Object**
Objects are used to store data in Salesforce. They are similar to tables in a database.

**Types of Objects:**
Standard Objects
These are provided by Salesforce.
Examples: Account, Contact, Opportunity

**Custom Objects**
These are created based on business needs.
Examples: Student, Course, Patient

Objects contain records and fields.

**Tab**
A Tab is used to open and access objects or pages inside an app. Tabs help users navigate through Salesforce easily.

**Examples:**
Accounts Tab
Contacts Tab
Students Tab

**CRM Concepts in Salesforce**

CRM concepts fit into Salesforce using objects.

Account represents a company or customer.
Contact represents a person.
Opportunity represents a sales deal or business opportunity.

These objects are grouped together inside apps to support business operations.

Configuration vs Coding
Configuration

Configuration means creating features without writing code. It is mainly used for simple automation and customization.

**Used for:**
Creating fields
Validation rules
Flows and automation
Page layouts and security settings

**Examples:**
Creating a custom object
Building automation using Flow Builder

**Coding**
Coding is used when advanced or complex functionality is required.

**Salesforce developers mainly use:**
Apex
Lightning Components
APIs

**Examples:**
Complex business logic
Integration with external systems

**Multi-Tenant Architecture**
Salesforce follows a multi-tenant architecture, where multiple customers share the same infrastructure while keeping their data secure and separate.
**Benefits:**
Scalability
Automatic updates
Lower maintenance

**How Developers Extend Salesforce**
Developers can extend Salesforce functionality using:
Apex
APIs
Lightning Web Components

**Integrations**
This allows businesses to customize Salesforce according to their requirements.
